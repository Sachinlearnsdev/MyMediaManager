Root Paths ?Three root directories define where everything lives. Manager Root is for code/config (SSD). Data Root is for processing pipeline (fast drive). Library Root is your final media library (Jellyfin/Plex reads from here).

Data Root and Library Root must be on the same drive so files move instantly (rename, not copy+delete). Manager Root stays on your SSD for code, cache, and logs. Sonarr/Radarr download folders should also live on the same drive as Data Root.
Different drives detected! Data Root and Library Root must be on the same physical drive. Cross-drive moves are slow (copy+delete) and can stall the pipeline. Move them to the same drive.
