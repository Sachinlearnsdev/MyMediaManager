Setup Guide

All set! MyMediaManager is fully configured and ready to use. Go to the Dashboard to start your services.

How MyMediaManager Works

1
Drop files

Put TV shows in Drop_Shows/ and movies in Drop_Movies/. Sonarr/Radarr can be configured to drop here automatically.

2
Pipeline processes

Files are automatically stabilized, extracted, routed, cleaned, classified, and matched against TVDB/TMDB/MAL APIs.

3
Library output

Matched files are renamed and sorted into 10 library categories, ready for Plex/Jellyfin to scan.

Drop Folder Mapping

Drop FolderCategories
Drop_Shows/ TV Shows, Cartoons, Anime Series, Reality TV, Talk Shows, Documentary Series
Drop_Movies/ Movies, Anime Movies, Stand-Up Specials, Documentary Films

Permissions (Bare Metal)

MyMediaManager uses the *arr pattern: a shared media group that Plex, Sonarr, Radarr, and MMM all belong to.

sudo usermod -a -G media plex
sudo usermod -a -G media sonarr
sudo usermod -a -G media radarr

All files are created with 775 permissions (group-writable) so any app in the media group can read/write them.
